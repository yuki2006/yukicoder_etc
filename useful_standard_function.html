細かく説明すると大変なので、簡単な説明だけでも。(現在運営者のメモ中)

C++

next_permutation
言わずと知れた、順列を求める事ができる関数。
ソートしておかないと、全ての順列を列挙できないことに注意。
prev_permutaionなどもある。
max_element, min_element
最大値・最小値を求める。
count
ある値の数を数える。
accumulate
総和を求める。和の型は要素の型でなく第3引数の型となるのでオーバーフローに注意。
unique
sort(v.begin(), v.end()); v.erase(unique(v.begin(), v.end()), v.end());
というコードにより、重複した要素を削除できる。
lower_bound, uppper_bound
lower_bound(a.begin(), a.end(), val): $a[i] \ge val$ である最左の位置
upper_bound(a.begin(), a.end(), val): $a[i] > val$ である最左の位置
を求める。
bitset
数値をビット集合として格納する。ビット演算は定数倍高速化が見込める

Java

PriorityQueue
優先度付きキュー https://docs.oracle.com/javase/jp/7/api/java/util/PriorityQueue.html
BigInteger
多倍長整数型クラス、modPowやmodInverse,gcdなどの便利メソッドもある。 https://docs.oracle.com/javase/jp/7/api/java/math/BigInteger.html
BitSet
C++の節にもあるがビットベクトルを生成する。

Python

heapq
優先度付きキュー 使い方としたら、専用のクラスがあるというわけでなく配列をヒープ化する。
Counter
連想配列とカウンターを持ったクラス 
        from collections import Counter
でimportする。
itertools.permutations
いわゆる順列、競プロによく出てくるあれです。
